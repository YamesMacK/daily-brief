# Daily Brief

Personal morning news brief — weather, markets, world, and construction.

Hosted at: https://yamesmack.github.io/daily-brief/

Refreshed automatically every morning at 6:00 a.m. CT by a scheduled Cowork task on James's PC. The task regenerates `daily-news.html` in the local outputs folder, copies it to `index.html`, and pushes the change here. GitHub Pages serves the file as a Progressive Web App — installable on phone home screens.

Source files:

- `index.html` — the news brief (regenerated daily)
- `manifest.webmanifest` — PWA manifest
- `icon-*.png`, `apple-touch-icon.png`, `favicon-32.png` — app icons
